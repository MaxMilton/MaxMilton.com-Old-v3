## How To Build

### For Development

Install prerequisites then run gulp to start a local dev server.

1. Install [Node.js](https://nodejs.org/)
2. Install [Hugo](https://github.com/spf13/hugo/releases)
2. `# cd themes/mm`
2. `# npm install`
3. `# gulp`

### For Production

Run build script

1. `# build.sh`
